# Public Assets Folder

Place your static assets (images, fonts, etc.) in this folder.

## For the Login Banner Image

1. Copy your coastal landscape image file to this folder
2. Name it something like: `login-banner.jpg` or `login-banner.png`
3. The image will be accessible at: `/login-banner.jpg` (or whatever you name it)

## Supported Image Formats

- JPG/JPEG
- PNG
- WebP
- SVG
